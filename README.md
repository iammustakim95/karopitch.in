# 🚀 Karo Pitch

> **Pitch Your Startup to India's Top Investors.** > A premium, modern, and highly interactive landing page designed for the Karo Pitch startup ecosystem platform.

🌍 **Live Demo:** [Visit Karo Pitch](https://iammustakim95.github.io/karopitch.in/)

---

## ✨ Features

This landing page was built with a focus on modern UI/UX principles, performance, and an investor-focused aesthetic. 

* 🌗 **Seamless Light & Dark Mode:** Fully adaptive color palettes with a beautiful toggle that remembers your preference using `localStorage`.
* 🪟 **Premium Glassmorphism:** Frosted glass navigation bars (`backdrop-blur`) and translucent UI cards that interact with the background.
* 🎨 **Creative Background Animations:** Smoothly animated floating gradient orbs and an infinite scrolling blueprint grid.
* 📱 **Fully Responsive:** A mobile-first approach ensuring the platform looks flawless on phones, tablets, and massive desktop monitors.
* 🅰️ **Modern Typography:** Uses **Outfit** for punchy, futuristic headings and **Plus Jakarta Sans** for crisp, highly readable body text.
* ⚡ **Scroll Animations:** Vanilla JavaScript Intersection Observers trigger smooth fade-in and hover-tilt effects as you scroll down the page.

---

## 🛠️ Tech Stack

This project is lightweight and fast, built without the need for heavy node modules or complex build steps:

* **HTML5:** Semantic and accessible structure.
* **Tailwind CSS (via CDN):** Rapid utility-first styling for complex gradients, flex grids, and dark mode variants.
* **Vanilla JavaScript:** For theme toggling, scroll events, and intersection observers.
* **Phosphor Icons:** Clean, consistent iconography used throughout the UI.
* **Google Fonts:** Hosted font families for optimized loading.
